# Elan Levin: UX Portfolio

A static, multi-page portfolio site: Home, Projects, About, and Contact.
Plain HTML/CSS/JS, no build step.

**Live site:** https://elanlevin1.github.io/portfolio/

## Structure

```
index.html       Home
projects.html    Case studies (Little Lemon, MacDaily, MSCS) plus
                 additional work (Path to Tranquility, Switch Adapted
                 Water Gun, Glistening Gay Terror Train)
about.html       Bio, working style, skills
contact.html     Contact details and form
styles.css       Shared design system
assets/          Project images, résumé, favicon
```

## Running locally

No build step required. Open `index.html` directly in a browser, or serve
the folder with any static file server, e.g.:

```bash
python3 -m http.server 8000
```
